Movie API Documentation

Introduction

Welcome to the documentation for our SpookyMovies API! This API provides access to a collection of movies along with their details, including title, summary, release date, genre, director information, image URL, and feature status.

Getting Started

To get started using the Movie API, follow these steps:

  1. Ensure you have access to the MongoDB database named "mySpookyFlixDB" where the movie data is stored.
  2. Clone or download the "movie_api" directory from our repository.
  3. Set up the necessary dependencies and environment variables as specified in the README file.
  4. Start the API server using the provided commands.

Endpoints

The Movie API provides the following endpoints:

  • GET /movies: Retrieves a list of all movies in the database.
  • GET /movies/:id: Retrieves details for a specific movie based on its ID.

Example Requests

Here are examples of how to make requests to the API:

    
      // Get all movies
      GET /movies
      
      // Get details for a specific movie (replace :id with actual movie ID)
      GET /movies/:id

Response Format

The API returns JSON-formatted responses with the following structure:

    
      {
        "Title": "Movie Title",
        "Summary": "Movie summary goes here.",
        "Release Date": "YYYY-MM-DD",
        "Genre": {
          "genreName": "Genre Name",
          "genreDescription": "Description of the genre."
        },
        "Director": {
          "directorName": "Director Name",
          "Bio": "Director biography goes here.",
          "Birth": "YYYY",
          "Death": "YYYY (if applicable)"
        },
        "imgUrl": "example.com/movie-image.jpg",
        "Feature": true
      }

Additional Notes

  • Make sure to handle errors gracefully, and provide meaningful error messages in the response.
  • Refer to the API documentation for more detailed information on request parameters and error codes.
